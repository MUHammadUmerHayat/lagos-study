Open Andela FAQs

  • Is this an Internship? This is not an internship, it is a training program
  • What is LASRRA?LASRRA is the Lagos State Residents Registration Agency
  • How do I register for LASRRA?Please go to the LASRRA home page to get more information on registration process
  • Do I need to reside in Lagos State to get the LASRRA Identification Number?Unfortunately, you do need to reside in Lagos to receive the LASRRA ID number.
  • Is the training program free?Yes, the program is free.
  • Is the program running during the week and weekends?The program only runs during the week
